# Crack Spalling Segmentation Dataset

## Overview
This dataset combines multiple existing crack segmentation datasets (crack-seg, HRCDS, S2DS) with newly collected field data and AI-generated synthetic samples. All annotations are formatted for YOLO-seg instance segmentation.

## Dataset Sources
- **Public Datasets**: 
  - [crack-seg](https://github.com/Wangyupei/Crack-Seg)
  - [HRCDS](https://doi.org/10.1016/j.engappai.2020.103762)  
  - [S2DS](https://s2ds.github.io/)
- **Self-collected Data**: Field images from real infrastructure
- **AI-generated Data**: Synthetic crack/spalling images

## Format
YOLO-seg (instance segmentation format)
